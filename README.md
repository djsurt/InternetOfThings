# Projects summary

I created this project as part of an Internet of Things class. Here is all the essential code that I wrote, debugged, and tested. 

LAMPI is a smart lamp device thats capable of adjusting its color, and brightness through user controlled interfaces. These user interfaces include a web interface, a capacitive touchscreen, and an iPhone. Technologies used include: AWS EC2, Django, Bluetooth Low Energy, MQTT, KivyUI, pigpio, and Swift.
